# PortScanner

### Java 多线程端口扫描器

有如下功能：
* 可以通过对给定的网址、IP地址或地址段进行扫描，扫描对应网站的开放端口号信息。
* 查看本机的IP地址。
* 将扫描结果以文件形式保存到本地。
